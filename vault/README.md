# SEO Agency Vault

This is an Obsidian-compatible knowledge vault. Every piece of SEO data, research, and deliverable lives here.

## Structure

| Folder | What Goes Here | Written By |
|--------|---------------|------------|
| `00-Dashboard/` | Agency dashboard, ROI tracker | CLAUDE.md (after every major deliverable) |
| `01-Clients/` | Client profiles, site info, competitors, goals, brand voice | Onboarding |
| `02-Audits/` | Master audit reports, compiled findings | Audit & Recommendations Team |
| `03-Research/` | Keywords, market research, trends, algorithms, SERP analysis | Research Team, Keywords Team |
| `04-Content/` | Content strategy, briefs, drafts, published, meta-tags | Content SEO Team |
| `05-Links/` | Backlink profiles, prospects, outreach, placements | Link Building Team |
| `06-Competitors/` | Competitor profiles, comparisons, monitoring | Competitor Analysis Team |
| `07-Reports/` | Monthly reports, quarterly reviews, dashboards | Analytics & Reporting Team |
| `08-Strategy/` | Roadmaps, priority matrices, growth plans | Strategy & Direction Team |
| `09-Local/` | GBP data, citations, local rankings, reviews | Local SEO Team |
| `10-Technical/` | Crawl data, Core Web Vitals, schema, redirects | Technical SEO Team |
| `11-AEO/` | AI Overview data, entity maps, citation tracking | AEO Team |

## Client Subfolders

Each client gets subfolders across all relevant sections:
```
vault/[section]/[client-name]/
```

## Conventions

- **Frontmatter** on every file (client, agent, team, date, type, status, tags)
- **Wikilinks** to connect related documents: `[[Company Name]]`, `[[keyword-clusters-2026-04-15]]`
- **Tags** for filtering: #audit, #keywords, #content, #links, #local, #technical, #aeo, #competitor, #report
- **Dates** in filenames: `[type]-[YYYY-MM-DD].md`

## Opening in Obsidian

Open the `SEO Agency In a Box` folder as an Obsidian vault. The `.obsidian/` config is included with graph view settings that color-code by section.
